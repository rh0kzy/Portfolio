Gestion de Cabinet Médical

Java Swing SQL JDBC

Vue d'ensemble du Projet

Une application complète de gestion de cabinet médical développée en Java avec une interface graphique Swing. Le système permet de gérer efficacement les dossiers des patients, les rendez-vous et les consultations médicales.

Structure du Projet

  • Architecture
    • Interface graphique : Swing
    • Base de données : SQL
    • Connexion : JDBC
    • Gestion des données : Java
  • Organisation des Dossiers 
    src/
├── model/             # Modèles de données
├── view/             # Interface graphique
├── controller/       # Gestion des interactions
├── dao/              # Accès aux données
├── utils/            # Utilitaires
└── main/             # Point d'entrée

Fonctionnalités Principales

  • Gestion des dossiers patients
  • Planning des rendez-vous
  • Suivi des consultations
  • Gestion des prescriptions
  • Statistiques et rapports

Technologies Utilisées

  • Java pour le développement
  • Swing pour l'interface graphique
  • SQL pour la base de données
  • JDBC pour la connexion
  • Programmation Orientée Objet
