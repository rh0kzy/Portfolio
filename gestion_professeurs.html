Gestion des Professeurs

React JavaScript HTML5 CSS3

Vue d'ensemble du Projet

Une application web développée avec React pour gérer les professeurs, les modules, les administrateurs et la charge de travail dans un établissement d'enseignement.

Structure du Projet

  • Architecture
    • Frontend : React.js (version 19.1.0)
    • Routing : React Router DOM (version 7.5.0)
    • UI Components : Composants personnalisés avec CSS
    • Dépendances principales : html2canvas, jspdf, lucide-react, react-icons
  • Organisation des Dossiers 
    src/
├── components/         # Composants réutilisables
├── Teachers/          # Gestion des professeurs
├── Workload/          # Gestion de la charge de travail
├── Admins/            # Gestion des administrateurs
├── Modules/           # Gestion des modules
├── Organigramme/      # Structure organisationnelle
├── Dashboard/         # Tableau de bord principal
├── Espace/            # Espace personnel
├── Login/             # Page de connexion
├── assets/            # Ressources (images, logos)
└── App.js             # Point d'entrée de l'application

Fonctionnalités Principales

  • Gestion des professeurs et de leurs informations
  • Suivi de la charge de travail
  • Gestion des modules et des cours
  • Interface administrateur
  • Génération de rapports et statistiques

Technologies Utilisées

  • React.js pour le frontend
  • JavaScript (ES6+)
  • HTML5 et CSS3
  • React Router pour la navigation
  • Bibliothèques UI modernes
